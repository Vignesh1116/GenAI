# Generative AI Fundamentals

# 1. What is Artificial Intelligence (AI)?

### Artificial Intelligence (AI) is a branch of computer science that enables machines to perform tasks that normally require human intelligence.

### Examples
Face recognition
Voice assistants (Siri, Alexa)
Self-driving cars
Chatbots
Recommendation systems

### Goal: Make machines think, learn, and make decisions like humans.

# 2. What is Machine Learning (ML)?

### Machine Learning is a subset of AI where computers learn patterns from data instead of being explicitly programmed.

### Example

Suppose you want to identify whether an email is spam.

Instead of writing hundreds of rules, you train the model with thousands of spam and non-spam emails. The model learns the patterns and predicts new emails.

### ML Workflow
Data
   ↓
Training
   ↓
Machine Learning Model
   ↓
Prediction
Types of Machine Learning
Supervised Learning
Unsupervised Learning
Reinforcement Learning

# 3. What is Deep Learning?

### Deep Learning is a subset of Machine Learning that uses Artificial Neural Networks with many layers.

### It is especially useful for:

Image Recognition
Speech Recognition
Language Translation
ChatGPT
Self-driving cars

### Why Deep Learning?
It automatically learns complex features without manual feature engineering.

# 4. What is Generative AI (GenAI)?

### Generative AI is a type of AI that creates new content instead of only analyzing existing data.

### It generates:

Text
Images
Videos
Audio
Code

### Examples
ChatGPT → Text
GitHub Copilot → Code
DALL·E → Images
Sora → Videos

### Example

Prompt:

Write a Python program to reverse a string.

# 5. Fundamentals of Generative AI

### a) Training Data

GenAI learns from huge datasets like:

### Books
Websites
Articles
Research papers
Code
Conversations

### b) Neural Networks

### The model uses deep neural networks to understand language and patterns.

### c) Transformer Architecture

### Modern GenAI models use the Transformer architecture.

### It helps the model understand relationships between words in a sentence using Attention Mechanism.

### d) Tokens

### AI does not read entire sentences at once.

### It breaks text into tokens.

### e) Prompt

### A prompt is the instruction given to the AI.

### f) Output Generation

### The model predicts one token at a time until the response is complete.

# 6. What is an LLM?

### LLM stands for Large Language Model.

### It is a deep learning model trained on massive amounts of text data to understand and generate human language.

### Examples
GPT
Llama
Gemini
Claude
Mistral

LLMs can:

Answer questions
Summarize text
Translate languages
Generate code
Write emails
Create stories

# 8. What are Embeddings?

### Embeddings are numerical vector representations of words, sentences, or documents that capture their meaning.

### Instead of storing text directly, embeddings convert text into numbers.

# 9. What is a Vector Database?

### A Vector Database stores embeddings instead of plain text.

### It quickly finds vectors that are most similar to a query.
